# RubikverseApi
Proyecto personal para practicar desarrollo backend con Spring Boot creando una plataforma para gestionar cubos de Rubik, registrar entrenamientos, guardar tiempos, ver estadísticas y manejar usuarios con autenticación JWT. Incluye documentación con Swagger y una UI hecha en JavaFX
