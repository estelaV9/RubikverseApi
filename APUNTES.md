## 1. Inicialización del proyecto con Spring Boot
1. Accede a la página del [Spring Initializr](https://start.spring.io)

2. Configura el proyecto con los siguientes parámetros:  
<img width="520" height="229" align="right" style="margin-left:10px;" alt="spring_initializr" src="https://github.com/user-attachments/assets/bc13353b-c3ba-4317-a59f-09323364e9b9"/>

   - **Project**: Maven  
   - **Language**: Java  
   - **Spring Boot**: 4.0.0  
   - **Project Metadata**:
     - **Group**: com.rubikverse  
     - **Artifact**: RubikverseApi  
   - **Packaging**: JAR  
   - **Java Version**: 17  
3. Añade las siguientes dependencias:  
   - **Spring Web** (para crear APIs REST)  
   - **Spring Data JPA** (para persistencia de datos con Hibernate)  
   - **MySQL Driver** (para conectarte a la base de datos)  
4. Haz clic en **Generate** para descargar el proyecto.  
5. Descomprime el archivo descargado y ábrelo en **IntelliJ IDEA**.
